# Adi Menon – GitHub Pages Site

This folder contains a simple, admissions-friendly personal website.

## Files
- `index.html` – main content
- `style.css` – styling

## Publish on GitHub Pages (quick)
1. Create a GitHub repository named: `YOURUSERNAME.github.io`
2. Upload `index.html` and `style.css` into the repo (root).
3. Go to **Settings → Pages**
4. Source: **Deploy from a branch**
5. Branch: `main` / Folder: `/ (root)` → **Save**
6. Your site will appear at: `https://YOURUSERNAME.github.io`

## Edit content
Open `index.html` and update:
- email address
- resume link
- optional GitHub/LinkedIn links
- any project descriptions

Tip: keep personal info minimal.
